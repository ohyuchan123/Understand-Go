# Understand-Go

## What is Go?
Go는 Google이 개발한 오픈소스 프로그래밍 언어로, 간단하고 신뢰성 있는 소프트웨어를 효율적으로 구축하기 위해 설계되었습니다.

## 주요 특징
1. 동시성 프로그래밍
    - 고루틴(Goroutine)을 통한 경량 스레드 지원
    - 채널(Channel)을 이용한 통신 기반 동시성 처리
    - CSP(Communicating Sequential Processes) 모델 채택

2. 컴파일 및 실행
    - 빠른 컴파일 속도
    - 정적 타입 컴파일 언어
    - 크로스 컴파일 지원
    - 단일 바이너리 생성

3. 메모리 관리
    - 가비지 컬렉션 내장
    - 포인터 지원 (포인터 연산 제한)
    - 메모리 안전성 보장

4. 언어적 특징
    - 심플한 타입 시스템
    - 인터페이스 기반 다형성
    - 명시적 에러 처리
    - 내장 동시성 지원
